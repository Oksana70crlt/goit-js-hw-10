
## Technical notes

Навігація між сторінками (`index.html`, `1-timer.html`, `2-snackbar.html`) реалізована через partial `header.html`, який підставляється на етапі збірки.

Автоперевірка аналізує вихідний `index.html` без урахування підстановки partial-ів, тому посилання на сторінки завдань додатково продубльовані в `index.html` як адаптація під обмеження інструменту перевірки.
